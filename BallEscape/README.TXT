        Ball Escape
        ===========
Version: 1.2
Date: 2021-09-20
       
The game takes place in space. The goal of the game is to get to the rocket ship safely while avoiding the astroids. 
The playable character is a crab where you can collect the floating stars to accumulate points. The game ends when
you reach the spaceship.

How to play:

Use the arrows "up", "down", and "left" to move the Crab.
